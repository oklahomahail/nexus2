## üé® Usage Guidelines

### ‚úÖ Correct Usage

- **Background**: Always use on dark backgrounds (#0D0D12 or darker)
- **Clear Space**: Maintain one node radius around all edges
- **Minimum Size**: 32px height (screen) / 12mm (print)
- **File Format**: Use SVG for web, PNG for print at 300 DPI

### ‚ùå Incorrect Usage

- Don't place on light backgrounds
- Don't place on busy or photographic backgrounds
- Don't distort proportions (maintain aspect ratio)
- Don't recolor individual nodes
- Don't add effects or filters

## üñºÔ∏è Export Specifications

### For Web

- **Format**: SVG (preferred) or PNG with transparent background
- **Size**: Use native SVG sizing or 2x for retina displays
- **Color Space**: sRGB
- **Optimization**: Already optimized, no additional compression needed

### For Print

- **Format**: SVG (vector) or PNG at 300 DPI
- **Color Space**: RGB (not CMYK - logo uses screen-specific glow effects)
- **Minimum Size**: 12mm height
- **Bleed**: Add 3mm bleed for printed materials

### For Social Media

- **Profile Picture**: Use `nexus_icon_dark.svg` (1:1 ratio)
- **Cover Photo**: Use `nexus_tagline_lockup.svg` with dark background
- **Open Graph**: 1200√ó630px with logo centered on #0D0D12 background
