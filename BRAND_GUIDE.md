## 1. üî∑ Core Logo System

### Primary Lockup

**Logo + Wordmark**
Use this version for most placements: web headers, splash screens, and marketing materials.

- **Symbol**: 6-node glowing network forming a central nexus point
- **Wordmark**: "Nexus" set in Inter Tight SemiBold, white (#FFFFFF) or light gray (#E5E7EB)

### Secondary Lockup

**Symbol Only**
Use for icons, app badges, or minimalist contexts.

- Gradient glow from indigo ‚Üí cyan
- Center white core (#FFFFFF)
- Minimum clear space: height of one outer node around the mark

### Spacing & Scale

- **Clear Space**: The height of one node radius around all edges
- **Minimum Size**: 32px height (screen) / 12mm (print)
- **Max Width Usage**: Up to 75% of viewport width for hero areas

## 4. üñºÔ∏è Logo Usage Guidelines

### ‚úÖ Correct Usage

- Use glow logo on dark backgrounds only
- Maintain gradient fidelity (indigo ‚Üí cyan)
- Use white or very light text alongside

### ‚ùå Incorrect Usage

- Don't invert the glow (keep white core centered)
- Don't use on busy or photographic backgrounds
- Don't distort proportions
- Don't recolor the nodes individually
